# Contributing

Thank you for your interest in contributing to the AI Support Ticket Assistant project.

## How to Contribute

1. Fork the repository.
2. Create a new feature branch.
3. Make your changes.
4. Test the workflow in n8n.
5. Submit a Pull Request with a clear description of your changes.

## Reporting Issues

If you discover a bug, please use the provided GitHub Issue Templates to submit a detailed report.

## Feature Requests

Feature suggestions are welcome. Please use the Feature Request template to describe your idea and the problem it solves.

## Code Style

- Use descriptive commit messages.
- Keep documentation up to date.
- Test all workflow changes before submitting.
- Follow existing project structure and naming conventions.

Thank you for helping improve this project!
