# Installation

## Requirements

- Node.js
- n8n Community Edition
- Google Gemini API Key

---

## Installation

```bash
npm install -g n8n
```

Start n8n

```bash
n8n
```

Open

```
http://localhost:5678
```

Import

```
workflow/AI-Support-Ticket-Assistant-v1.json
```

Create a Google Gemini API credential.

Run the workflow.

---

## Configuration

Configure:

- Google Gemini API Key
- AI Model
- Prompt

The workflow is now ready.
