# Workflow Overview

## Nodes

### Manual Trigger

Starts the workflow.

---

### Edit Fields

Creates a simulated customer support request.

Fields:

- Customer Name
- Email
- Subject
- Message

---

### Basic LLM Chain

Processes the support request using Google Gemini.

---

### Google Gemini Chat Model

Provides AI-powered technical analysis.

---

## Output

The workflow generates:

- Issue Summary
- Priority
- Technical Assessment
- Possible Root Causes
- Troubleshooting Steps
- Escalation Recommendation
- Customer Response
